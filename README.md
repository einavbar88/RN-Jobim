# RN-Jobim

### Description
A replica of drushim.co.il's JOBIM app from 2021, it includes:
- REST API (Node.JS and Express) 
- DB in MongoDB
- Cross-Platform Mobile App in React Native [EXPO]

### Technologies
- React Native
- React Navigation
- Node JS
- MongoDB
- GoogleAPI
